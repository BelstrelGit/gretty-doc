2014-05-29
:sectanchors:
:jbake-type: page
:jbake-status: published

Gretty requires:

* JDK6, JDK7 or JDK8.

* Gradle 1.11 or newer.

* Enough operating memory and disk space to run gradle, jetty, tomcat.

* Access to jcenter repository or equivalent.

Support of JDK6 is coming soon.
